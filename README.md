## 🤔 这是什么？
它是一个工作流。可快速构建 带docker且支持自定义固件大小的 immortalWrt
> 1、固件大小 默认1GB <br>
> 2、无docker<br>
> 3、只生成x86-64 <br>
> 4、管理地址192.168.2.1 <br>
